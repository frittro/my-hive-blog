Remixing the Hive coin image from @bidesign with some standard emoji symbols to make new icons for my post header images.

Post Payout Options Icons.

My post head thumbnail images are created from a template, using GIMP. I am forever tweaking and improving my template for them. In this post I will explain a new feature of my post head thumbnail image generator template, which I will be using from this point forward.

You will notice in the top right corner of this post's thumbnail image, a new icon which represents the post's selected payout option. In this particular instance, the payout has been declined. Other posts may have an icon for accepted payout, burned payout, or donated payout.

Creating the remix.

These images were simple to put together in GIMP, using layers. Then I just turned off all the layers but the ones that I needed for each individual post payout option, and rendered the PNG image for that specific one. Each individual image was thus rendered by simply manipulating the layers of the XCF file.
